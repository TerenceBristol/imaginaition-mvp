## ğŸš€ Quick Start

### Prerequisites
- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/TerenceBristol/imaginaition-mvp.git
cd imaginaition-mvp

# Navigate to the Next.js project
cd imaginaition-website

# Install dependencies
npm install
```

### Build for Production

```bash
npm run build
npm start
```
